### ⚙️ 개발 환경
- `Java 11`
- `JDK 11.0.17`
- **IDE** : sts 4.18(Spring Tool Suite 4)
- **Database** : Mysql

## 📌 주요 기능
#### 로그인
- DB값 검증(인증처라)
- 로그인 시 쿠키(Cookie) 및 세션(Session) 생성
- 등록 개인 입사사진 가져오기
#### 마이페이지
- 출퇴근 기능 구현
- 나의 연봉 확인 
- 수정 가능한 개인정보 수정 기능
- 나의 직급,사번,부서
- 연도별 호봉에 따른 연봉 알고리즘
#### 사원등록
- 사번 중복 체크
- 입사 사진 등록 
#### 마이 페이지
- 내질문 확인하기

#### 1 대 1 질문
- 글 작성, 댓글 확인,  삭제(CRUD)

#### 관리자 페이지 
- 전체 질문 조회 답글달기

#### 관리자, 사용자 둘다 해당
- 어플리케이션 과 세션을 사용하여 질문을 답과 확인 삭제를 아이디 키값으로 받아서 유동적으로 확인하게 만듬
